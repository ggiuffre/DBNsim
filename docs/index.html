How to use DBNsim properly

What is DBNsim

DBNsim is a web application for training and analysing Deep Belief Networks ("DBNs", for short). A Deep Belief Network is a particular type of artificial neural network; it is a stack of Restricted Boltzmann Machines ("RBMs", for short) that can be trained with an algorithm known as "contrastive divergence". RBMs learn to reconstruct a given input and, if properly analysed, they can reveal latent features in the distribution of the input.

DBNsim offers a simple interface for defining the architecture of a DBN, train it on a remote web server (optionally equipped with a GPU), analyse it, and then download the DBN in various formats for performing further analyses.

How to run DBNsim on your machine

As said, DBNsim is a web app. That means you can run it on one computer and access it on another. According to you needs, there are several ways in which you can use DBNsim:

  • If you want to use it locally (that is, as if it was a normal desktop application on your PC), you just have to download Python and launch the app with the command line; then, you will be able to access the user interface on your browser at the local address http://127.0.0.1:8000/DBNtrain/. Read here for a step-by-step guide.
  • You may want your students to use it in a classroom. For this, you just have to launch the app with the command line; then ask your audience to connect their devices to the University network and to go to http://you-public-ip-address:8000/DBNtrain/. Read here for more details.
  • Finally, if you don't have a local network like above, you can still publish DBNsim on a remote web server: read here for understanding how to do this.

In all three cases, launching DBNsim from a computer equipped with a CUDA-enabled GPU lets you train a DBN directly on the GPU; DBNsim should automatically detect whether or not your computer has a GPU. The main advantage of using it is speed: you will be able to train much larger networks in fewer time.
